# Frontend (React + Vite)

Production UI for the property valuation flow.

## Setup

```bash
npm install
cp .env.example .env   # VITE_API_URL=http://localhost:8001
npm run dev
```

Or from the repo root: `make frontend` / `make dev`.

## Stack

React 19, Vite, Tailwind, shadcn/ui, Leaflet, i18n (es/en).
