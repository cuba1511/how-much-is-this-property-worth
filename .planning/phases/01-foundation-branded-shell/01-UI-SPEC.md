## Spacing Scale

Declared values (multiples of 4). Already configured in `tailwind.config.js` → `theme.extend.spacing`:

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, gap between badge items |
| md | 16px | Default element spacing, card internal padding-x |
| lg | 24px | Section padding, gap between hero text elements |
| xl | 32px | Layout gaps, hero section vertical padding |
| 2xl | 48px | Major section breaks, hero top padding on desktop |
| 3xl | 64px | Page-level spacing, bottom padding below placeholder card |

Exceptions: 12px (`mt-3`) used for headline-to-subtitle gap — inherited from existing `h1` CSS rule (`margin: 24px 0 12px`). Retained for visual tightness between headline and subtitle; falls outside the 8-point scale but is a deliberate reuse of the established typographic margin.

## Typography

Font: Inter via CSS variable `--sans`. Already set in `index.css` `:root` block with `font: 18px/1.45 var(--sans)`.

| Role | Size | Weight | Line Height | Tailwind / CSS |
|------|------|--------|-------------|----------------|
| Display (h1) | clamp(2rem, 4vw + 1rem, 3.25rem) | 600 (semibold) | 1.1 | `h1` selector in `index.css`, `letter-spacing: -0.03em` |
| Heading (h2) | 1.35rem (21.6px) | 600 (semibold) | 1.2 | `h2` selector in `index.css`, `letter-spacing: -0.02em` |
| Body | 18px desktop / 16px ≤1024px | 400 (regular) | 1.45 | `:root` font declaration with media query |
| Label | 14px | 600 (semibold) | 1.4 | `text-sm font-semibold` |

Weights used: 400 (body text, subtitle) and 600 (headings, labels, CTA button).

Source: `frontend2/src/index.css` lines 72–85 (`:root` font declaration + responsive breakpoint).

## Color

### 60/30/10 Split

| Role | Value | Tailwind Token | Usage |
|------|-------|----------------|-------|
| Dominant (60%) | #ffffff | `bg-surface` / `bg-background` | Page background, hero section background |
| Secondary (30%) | #f3f5fe | `bg-surface-tint` | Placeholder card background, form mount area |
| Secondary alt | #f5f7f9 | `bg-surface-muted` | Available for subtle section differentiation (not used in Phase 1) |
| Accent (10%) | #f45504 | `bg-accent` / `.btn-primary` | CTA button ONLY — "Valorar mi propiedad" button |
| Primary | #2050f6 | `text-primary` | Headline accent text ("tu casa" in electric blue), link color |
| Destructive | hsl(0 84% 60%) | `bg-destructive` | Not used in Phase 1 — reserved for future error states |

### Accent Reserved For

Accent (#f45504 orange) is **exclusively** for:
1. Primary CTA button background (`.btn-primary`)
2. CTA button hover state (#ff7a40 via `hover:bg-accent-light`)

Accent is **NOT** used for: links, icons, borders, badges, headings, or decorative elements.

### Primary Blue Usage

Primary (#2050f6 electric blue) is used for:
1. Headline accent text — the words "tu casa" in the h1
2. Focus ring outlines (`focus-visible:outline-primary`)
3. Subtle border tint on `#root` container (`border-inline: 1px solid var(--border)`)

Source: CONTEXT.md D-01, D-02. Tokens from `tailwind.config.js` and `index.css`.

## Layout Contract

### Responsive Breakpoints

| Breakpoint | Width | Behavior |
|------------|-------|----------|
| Mobile | < 768px | Single column, reduced padding (md: 16px horizontal), h1 scales down via clamp, font-size drops to 16px |
| Tablet | 768px–1024px | Same as mobile layout but with more horizontal breathing room |
| Desktop | > 1024px | Full 1126px container, 48px top padding on hero, 18px body font |

### Root Container

- Max width: 1126px (existing `#root` CSS)
- Centered: `margin: 0 auto`
- Border: subtle `border-inline: 1px solid var(--border)` (existing)
- Min height: `100svh` (existing)
- Flex column layout (existing)
- **Change**: Remove `text-align: center` — hero content is left-aligned

### Focal Point

**Primary focal point:** `h1` headline — largest element on the page (clamp 32–52px), first visual anchor on load. The electric blue "tu casa?" accent text provides a secondary color focal point within the headline.

### Vertical Rhythm

```
[Page top]
  └─ HeroSection
      ├─ padding-top: 48px (2xl)
      ├─ padding-x: 32px (xl) desktop / 16px (md) mobile
      ├─ h1
      ├─ gap: 12px
      ├─ subtitle
      ├─ gap: 24px
      ├─ CTA button
      └─ padding-bottom: 32px (xl)
  └─ gap: 32px (xl)
  └─ PlaceholderCard
      ├─ margin-x: 32px (xl) desktop / 16px (md) mobile
      ├─ padding: 48px (2xl)
      ├─ icon + placeholder text (centered)
      └─ margin-bottom: 64px (3xl)
[Page bottom]
```
